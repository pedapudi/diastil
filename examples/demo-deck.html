Premise

The saved file is plain HTML

This deck is a self-contained HTML file. Open it anywhere; it presents itself.

Every style reads a token from the theme block, so one edit restyles the deck.

Any agent or text editor can keep working on it — there is no private format.

Ingest pipeline

Every deck compiles to a verified dialect

Foreign decks execute in a sandbox; the rendered DOM is ground truth.

Translation is schema-constrained; content is copied, never regenerated.

A visual compare proves the conversion, or the region is preserved verbatim.

Typesetting

Roles bind to a scale, not to pixels

Titles are step 5. Kickers are step 1, tracked wide. Body is step 2 on a 1.55 leading.

Build steps are declarative attributes — this line reveals second in present mode.

And this one third. No script in the document expresses that; the runtime interprets it.

Photosetting

Images carry crop and focal point as style

The figure's focal point is data, so exports never re-encode pixels.

Shapes

Shapes are nodes that haven't spoken yet

A circle or square is a label-less node: drag it, resize it (⇧ keeps it true), connect it, or double-click to give it words.

Fill, line, and weight are token-scoped properties on the shape itself — restyle one shape without touching the theme.
