Six metaphor seeds from the diastil house style, drawn to the same economy: faint ink-faint structure with layered opacity, dashed rule guides, and exactly one accented element — the thing the slide argues. Paper theme, Source faces.

full color — the licensed exception

When the subject wants a real palette, artwork may go beyond the theme tokens. Each piece OWNS a palette family (dawn, twilight, sandstone, slate-green) AND its own setting — a landscape, an object study, an interior, a machine — so a deck's pictorial figures vary in both color and staging instead of repeating one look. The grammar stays: flat facets, atmospheric recession, and one dotted accent (recolored per ground for contrast) that is the argument.
